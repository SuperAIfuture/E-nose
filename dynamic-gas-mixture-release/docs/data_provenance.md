# Data Provenance

- Benchmark source: UCI Machine Learning Repository, Gas Sensor Array Under Dynamic Gas Mixtures.
- Repository content: processed shards required for the released runs, plus metadata for the omitted raw files.
- Omitted files: the original archive and extracted raw text files exceed standard GitHub size limits and are therefore documented in `data/raw/raw_data_manifest.json` instead of being committed directly.
- Processing metadata: `data/processed/uci_dynamic_mixtures_322_v2/ethylene_CO/meta.json`.

Users should verify and follow the original benchmark terms when redistributing the raw source files separately.
