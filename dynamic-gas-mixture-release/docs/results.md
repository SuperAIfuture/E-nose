# Results Layout

## Main
- `results/main/summary.json`: aggregate metrics and selected operating point.
- `results/main/results_tables.csv`: method-level summary table.
- `results/main/per_seed_gate_table.csv`: seed-wise gate outcomes.
- `results/main/per_seed_outputs/`: detailed per-seed JSON outputs.

## Extensions
- `results/extensions/fixed_config_30seed/`: fixed-configuration 30-seed robustness extension.
- `results/extensions/ungated_ablation_10seed/`: ungated ablation outputs.
- `results/extensions/temporal_baselines_10seed/`: additional temporal baseline outputs.
