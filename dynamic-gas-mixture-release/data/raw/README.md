# Data Layout

## Processed Data
- `data/processed/uci_dynamic_mixtures_322_v2/ethylene_CO/`
- Includes `meta.json` and `shard_*.npz`.
- The processed dataset is sufficient to rerun the released scripts in this repository.

## Raw Data
- The original raw archive and extracted text files are not committed into this GitHub-ready repository because they exceed standard GitHub file-size limits.
- File metadata and checksums are recorded in `data/raw/raw_data_manifest.json`.
- If needed, these raw files can be distributed separately through GitHub Releases, Zenodo, or another large-file host.
